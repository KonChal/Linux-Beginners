### Not in alphabetical order!

| Command | Arguments | Description |
| :---         |     :---:      |          ---: |
| cd   |        | Change the directory |
| pwd     |       | Output of the current working path      |
| ls     |       | List a folder content      |
| ls     |   -l    | Long Listing    |
| ls     |   -a    | Displays everything even hidden files     |
